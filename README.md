> ## Statistics

통계와 관련된 강의, 책, 영상 등의 공부한 내용들을 지식의 원천의 제목과 함께 요약해놓는 요약 모음 repo

---

| 필수 요건 |                                                                                    |
| :-------: | ---------------------------------------------------------------------------------- |
| 전체 제목 | 강의라면 강의명, 책이라면 큰 제목, 영상이라면 큰 주제명 etc.                       |
|   저자    | 강의 교수, 책의 저자, 영상 속 설명자 and so on                                     |
|   링크    | 위의 제목, 저자와 같이 강의를 특정하기 위한 수단으로 site or 책 (판매류) 소개 링크 |
